# 🚀 Quick Start - Trading Dashboard v3.0

Get your trading analytics running in **2 minutes**!

---

## ⚡ Start the Dashboard

### 1. Start Backend
```bash
cd backend
uvicorn main_position:app --reload --port 8002
```

### 2. Start Frontend
```bash
cd frontend
npm run dev
```

### 3. Access Dashboard
```
http://localhost:5173
```

---

## 📊 Upload Your Trades

### Single File (Auto-Detect)
1. Click **"Single File"** mode
2. Upload CSV (stocks or options)
3. View dashboard

### Combined (Stocks + Options)
1. Click **"Stocks + Options"** mode
2. Upload both CSVs
3. View unified metrics

---
